# Security Policy

## Supported Versions

We actively support and fix vulnerabilities in the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability within this repository, please do **NOT** open a public issue. Instead, report it privately to the maintainers to protect user installations.

Please send vulnerability details directly to:
- **Email:** security@example.com (Replace with actual email)

Include:
1. Steps to reproduce the issue.
2. A description of the impact.
3. Any potential mitigations you've identified.

We will acknowledge your report within 48 hours and work to resolve the issue as quickly as possible.
