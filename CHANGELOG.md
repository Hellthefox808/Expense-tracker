# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-19

### Added
- Created complete MVC directory structure separating models, views, controllers, and routes.
- Configured dedicated database connection file under `config/db.js` with auto-reconnections.
- Integrated payload validation middleware and a global boundary error handler.
- Integrated Helmet Content Security Policy policies and express-rate-limit protection.
- Designed premium Ethereal Shadows animated SVG liquid fluid background and fine static noise overlay.
- wired requestAnimationFrame mouse tracking coordinates to radial gradient light effects.
- Overhauled the home page into a premium obsidian grid dashboard.
- Installed top-level summary metrics (Total spent, count, peak expense).
- Added Custom radio pills category selector and interactive filter pills.
- Added visual Analytics progress bars to Doughnut chart.
- Constructed a complete Jest and Supertest unit & integration test suite.
- Provided Docker containerization via Dockerfile and docker-compose.
- Configured GitHub Actions CI pipeline, Dependabot policies, and CODEOWNERS.
