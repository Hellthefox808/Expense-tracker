# AI Engineering & Agent Collaboration

This project has been developed in pair-programming collaboration with **Antigravity**, an agentic AI assistant designed by Google DeepMind.

---

## Agentic Development Workflow

We utilized agentic workflows to refactor, secure, and polish this codebase:

1. **MVC Refactoring:** Evolved a single-file server script into a modular MVC structure.
2. **Security Hardening:** Audited and patched Content Security Policies and rate limiting.
3. **Premium UI/UX Iteration:** Crafted custom SVG displacement filters and mouse-tracking animations.
4. **Integration Testing:** Created complete Supertest assertions to verify endpoints automatically.

---

## Prompt Patterns used

- **Visual Feedback Loop:** Utilizing browser subagents to verify rendering layouts and CSS styles without human intervention.
- **Resilient Refactoring:** Performing compile-time syntax validations (`node -c`) before submitting changes.
